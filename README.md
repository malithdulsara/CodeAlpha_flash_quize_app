# Flashcard Quiz App 🧠

A simple, interactive flashcard application built with HTML, CSS, and vanilla JavaScript to help users study efficiently. 

## 📝 Description

This project is a customizable study tool that allows users to create their own flashcards. It features a clean, intuitive interface with smooth 3D flip animations to reveal answers. Users have full control over their study material, with the ability to add new cards, update existing ones, or remove ones they have already mastered.

## ✨ Features

* **Interactive Flashcards:** Each card displays a question on the front and an answer on the back, complete with a "Show Answer" button and click-to-flip functionality.
* **Easy Navigation:** Quickly move through your study deck using the "Next" and "Previous" buttons.
* **Customizable Deck (CRUD):** 
  * **Add:** Create new flashcards directly from the user interface.
  * **Edit:** Modify existing questions and answers.
  * **Delete:** Remove cards you no longer need.
* **Clean UI:** A simple, modern, and responsive user interface designed for distraction-free studying.

## 🛠️ Technologies Used

* **HTML5:** For the structure and layout of the app.
* **CSS3:** For styling, layout, and 3D flip animations.
* **Vanilla JavaScript:** For DOM manipulation, state management, and handling user interactions.
